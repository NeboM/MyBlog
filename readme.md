## Blog application
###Quick start

1. Rename .env.example to .env
2. Set up database configuration in .env file
3. *composer install*
4. *php artisan migrate*
5. *php artisan db:seed*
6. *php artisan key:generate*
7. *php artisan storage:link*
8. *php artisan serve*
